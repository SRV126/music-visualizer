Music Visualiser Application

Steps to run the program.
1. Open Visual Studio 
2. Create a new Win32 Application Project.
3. Make it a blank project.
4. Copy the source file to the project source directory.
5. Build Source, Debug the program.


Steps to use the program:

1. Run the program
2. Right click to open menu and choose options from there
3. Drag the mouse inside the window to change the design
4. All options like changing speed,visuals,autospin and exit are present in right click menu.
